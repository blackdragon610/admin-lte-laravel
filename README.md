## Running
1. composer install 
2. php artisan key:generate
3. npm run install
4. setting .env untuk konek ke database
4. php artisan migrate
5. php artisan db:seed
6. php artisan serve
7. npm run watch

## Note Jika Ingin Mengosongkan database
php artisan migrate:rollback
